# Dutch League Structure

The Dutch league structure is as follows:

- **Eredivisie**
- **Eerste Divisie**
- **Tweede Divisie**
- **O21 Leagues** (Divisie 1, Divisie 2, Divisie 3, Divisie 4 A-B) — these sit in the league table display above the Derde Divisie and under the Tweede Divisie
- **Derde Divisie** (A-B)
- **Vierde Divisie** (A-D)
- **Regional Structures:**
  - **Noord**
    - Eerste Klasse H
    - Zaterdag Noord
      - Tweede Klasse I-P
      - Derde Klasse Q-R
      - Vierde Klasse A-F
      - Vijfde Klasse A-G
    - Zondag Noord
      - Tweede Klasse G-H
      - Derde Klasse N-P
      - Vierde Klasse A-C
      - Vijfde Klasse A-D
  - **Oost**
    - Eerste Klasse F-G
    - Zaterdag Oost
      - Tweede Klasse G-H
      - Derde Klasse L-O
      - Vierde Klasse A-E
      - Vijfde Klasse A-E
    - Zondag Oost
      - Tweede Klasse F
      - Derde Klasse K-M
      - Vierde Klasse A-E
      - Vijfde Klasse A-G
  - **West 1**
    - Eerste Klasse A
    - West 1 Zaterdag
      - Tweede Klasse A-B
      - Derde Klasse A-D
      - Vierde Klasse A-E
      - Vijfde Klasse A-E
    - West 1 Zondag
      - Tweede Klasse A-B
      - Derde Klasse A-B
      - Vierde Klasse A-C
      - Vijfde Klasse A-C
  - **West 2**
    - Eerste Klasse B-C
    - West 2 Zaterdag (Zondag does not exist in this regional structure)
      - Tweede Klasse C-D
      - Derde Klasse F-G
      - Vierde Klasse A-E
      - Vijfde Klasse A-E
  - **Zuid 1**
    - Eerste Klasse D
    - Zuid 1 Zaterdag
      - Tweede Klasse E-F
      - Derde Klasse H-K
      - Vierde Klasse A-E
      - Vijfde Klasse A-D
    - Zuid 1 Zondag
      - Tweede Klasse C
      - Derde Klasse C-F
      - Vierde Klasse A-D
      - Vijfde Klasse A-F
  - **Zuid 2**
    - Eerste Klasse E
    - Zuid 2 Zondag (Zaterdag does not exist in Zuid 2)
      - Tweede Klasse D-E
      - Derde Klasse G-J
      - Vierde Klasse A-I
      - Vijfde Klasse A-G 

# Advanced Gameplay Features

## 1. Dynamic Player Development & Personalities
- Add player personalities (traits like ambitious, loyal, volatile) that affect morale, transfer requests, and development.
- Implement a mentoring system where senior players can mentor youth, boosting growth or changing personality.
- Add hidden attributes (e.g., professionalism, consistency) that affect performance and development.

## 2. Advanced Tactics & Match Engine
- Allow users to design custom tactics, set pressing, defensive lines, and player roles.
- Enable in-match tactical changes (real-time tweaks, shouts, substitutions).
- Show detailed match stats: xG, heatmaps, pass maps, player ratings.

## 3. Board & Fan Dynamics
- Implement board objectives (multi-year goals, consequences for failure).
- Add fan sentiment that affects board patience, ticket sales, and transfer budgets.
- Add media/press conferences with questions, rumors, and morale-impacting events.

## 4. Financial Depth
- Add sponsorship negotiations (sign, renew, lose sponsors based on performance).
- Enable stadium expansion, new builds, or city moves (with fan reactions).
- Add merchandise, TV, and prize money as income streams.

## 5. Scouting & Transfers
- Build a scouting network (assign scouts to regions, uncover hidden gems).
- Add transfer negotiations (agent fees, clauses, counter-offers).
- Expand loan system (wage splits, recall options, development promises).

## 6. Youth Academy & Regens
- Add youth tournaments (U21/U18 leagues and cups).
- Implement regens/newgens (retired stars "reincarnate" as new youth players).
- Enable academy upgrades (facilities and staff affect youth intake).

## 7. International Management
- Allow national team jobs (manage country, call-ups, tournaments).
- Handle international breaks (player fatigue, injuries, morale).

## 8. Procedural Events & World Evolution
- Make club reputations dynamic (rise/fall, takeovers, bankruptcies).
- Generate procedural news (stories, scandals, rumors).
- Add AI managers with personalities, tactics, and transfer strategies.

## 9. Online & Social Features
- Add online leaderboards (achievements, trophies, high scores).
- Enable multiplayer leagues (compete with friends).
- Allow shareable highlights (match highlights, achievements on social media).

## 10. Modding & Customization
- Build an editor for clubs, players, leagues, competitions.
- Add skin/mod support (custom logos, kits, UI themes).
